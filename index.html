如何创建属于自己的个人博客

发表于 2020-05-01 更新于 2020-11-02

在日常学习生活中，总有一些知识需要被记录下来。

我的博客搭建 hexo+github

一、安装必要的工具 node.js

安装node.js的目的主要是使用到它的npm包管理工具，通过命令行就可以通过指令去下载hexo框架

node.js的下载安装

直接去node.js官网下载系统对应的安装包，然后安装到本地即可。

因为我的系统是win10，安装完成之后

打开cmd命令行工具，任意目录下，输入以下指令，确认node.js的版本

1、node -v

2、npm -v

确认安装完成之后就可以安装hexo框架了。

由于国内npm的速度得不到保证，因此一般情况下，都会安装一个npm的淘宝镜像，这样就会使下载各类包的时候的速度能更快一点，具体操作如下：

npm install -g cnpm --registry=http://registry.npm.taobao.org

然后使用cnpm -v确认下载的cnpm的版本

之后就可以使用cnpm来代替npm了

二、安装hexo框架

1、cnpm install -g hexo-cli

安装hexo框架

2、hexo -v

查看hexo版本

如果出现上图类似的情况就说明安装成功。

三、创建保存博客的文件夹，并用hexo初始化博客

1、创建一个空的文件夹，然后再命令行中cd到这个目标文件夹

2、hexo init

如果提示 Start blogging with Hexo，则说明初始化完成

此时就可以在本例预览自己的个人博客了。

3、hexo start或hexo s

此时就可以打开浏览器，在地址栏输入localhost:4000,就可以看到自己的博客内容了。

但是，目前仅仅之只能本地预览，还无法达到随时随地查看博客内容的目的。

四、新建一篇新的文章

1、hexo new "文章标题"或hexo n "文章标题"

创建完成之后，接着就是清理hexo原先生成的文件

2、hexo clean

接着就是重新生成博客

3、hexo g

这样就成功新建了一篇新的博客了

五、将博客部署上线

接下来就是个人博客最终要的部分了，完成这个步骤之后就可以实现随时浏览博客的目的了

因为我用的是github来作为的个人博客的线上仓库

所以，接下来我将演示如何将博客部署到github之中

1、在github中新建一个仓库

注意：新建的仓库名必须按YourGithubName.github.io这种格式进行创建

2、在本地的博客目录下，安装git部署插件

cnpm install --save hexo-deployer-git

3、安装完成之后，还需要修改_config.yml配置文件，这个文件就在本地的博客目录之下

通过vscode打开，找到deploy的配置信息，修改成以下内容

修改type，将‘ ’改成git

添加repo，并将github中的仓库地址作为它的值

添加branch，如果不添加这个选项，则默认是master分支

4、部署到github仓库

hexo d

这样就部署成功了。此时我们就可以通过在地址栏输入github中的仓库名称，访问到自己的个人博客啦。

比如说: coderwuyf.github.io这就是我的github仓库名称，同时也是我的个人博客地址

六、补充：如何修改个人博客的主题

git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia #下载yilia主题到本地

#修改hexo根目录下的 _config.yml 文件 ： theme: yilia

hexo c #清理一下
hexo g #生成
hexo d #部署到远程Github仓库
https://coderwuyf.github.io #查看博客,成功~
